# XML → JSON Translator using Lex & Yacc

## Compilare

make

## Rulare

./translator < test1.xml

sau

make run

## Functionalitati

- parsare XML
- verificare tag-uri deschise/inchise
- conversie automata in JSON
- suport pentru elemente imbricate
- afisare structurata

## Tehnologii

- Lex/Flex
- Yacc/Bison
- C
