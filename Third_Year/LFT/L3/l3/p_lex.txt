1. Scrieti o specificatie in lex care sa permita urmatorul comportament. Daca la intrare se primeste un cod in C care include:
a) statement for
b) instructiuni printf
c) in aceste statements pot aparea variabile cu identificatori formati din litere mici/mari,_ si -, dar care incep cu o litera
d) numere intregi

returneaza o secventa de tokeni de tipul:
a) NUM - pentru numere
b) VAR pentru variabile
c) KEY - pentru keyword.
d) operatori =, +, separatori , {, }...
Exemplu:
for (i = 1; i < 11; i++)
  {
    printf("%d ", i);
  }
  return 0;

KEY ( var = NUM; var < NUM; VAR + + ) { KEY ( STRING  , VAR ) } KEY NUM ;
